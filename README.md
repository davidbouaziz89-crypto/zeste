# 🍋 Zeste — Tâches & Courses

Application web (PWA) ultra simple et colorée pour gérer ses **tâches à faire** et sa **liste de courses**, pensée pour iPhone.

## Fonctions
- ✅ **Tâches** classées par catégories personnalisables (Perso, Bureau, Maison, Santé, Famille…)
- 🔔 **Rappels** avec date/heure + répétition, notifications quand l'appli est ouverte
- 📅 **Ajout à Google Agenda** en un tap
- 🛒 **Courses** rangées automatiquement par rayon (~250 produits reconnus + emoji auto)
- 🔁 **Articles/tâches récurrents** (essentiels rachetés en 1 clic)
- ⚙️ **Catégories entièrement paramétrables** (nom, emoji, ordre)
- 📱 **Installable** sur l'écran d'accueil, fonctionne hors-ligne
